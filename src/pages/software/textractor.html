Textractor - Wordlist Creator

This is a command line C# .NET tool for extracting words from large files. It works on Windows with the .NET framework and on Linux with Mono. It is most useful when used against large, text-based databases to create wordlists. It's what I used to create part of crackstation.net's wordlists from the Wikipedia databases.
