Textractor

This is a command line C# .NET tool for extracting words from large files. It works on windows with the .NET framework and on Linux with Mono.

Usage

Usage: textracter [options] [filename]

STDIN will be used if no filename is specified.
If no options are specified, only alphanumeric strings longer than 3 characters and shorter than 64 characters are printed.

Options:
--minlen Minimum word length.
--maxlen Maximum word length.
--tolower Make all words lowercase.
--toupper Make all words UPPERcase.
--keeporiginal Print original case also when --tolower or --toupper is used.
--includeset <o> Specify a character set to include in output.
Options are: alpha, alphalow, alphahigh, space, numeric,
symbols, alphanumeric, ascii, unicode.
Can be used multiple times.
--excludeset <o> Specify a character set to exclude from output.
See --includeset for options.
--includechars <o> Includes the following characters in output.
--excludechars <o> Excludes the following characters from output.
-v Show # of lines processed every million lines on stderr.
-o <file> Output file. This is MUCH faster than piping to file.


Examples:

Get all alphanumeric strings:
textracter file.dat > wordlist.lst

Get all alpha and symbol strings:
textracter --includeset alpha --includeset symbols file.dat > words.lst

Make all words lowercase:
textracter --tolower file.dat > wordlist.lst

Make all words lowercase and keep original case (output both):
textracter --tolower --keeporiginal file.dat > wordlist.lst

Print all characters except for symbols:
textracter --includeset unicode --excludeset symbols file.dat > wrds.lst

Print only words made up of 'a', 'b' and 'c':
textracter --includechars abc file.dat > wordlist.lst
