Password Bolt is a new web based password manager. It was designed because there are currently no other open source web based password managers. It is still in it's peer review and development process, so a stable release is not available yet. Setting it up is just a matter of copying the php files and importing the SQL database. Feel free to download and try it out!

PasswordBolt is open source, so if you're an open source person, feel free to take this and improve it on your own because I use this personally but I don't have time to maintain it as an open source project.

Highlights

  • AES-TWOFISH-SERPENT encryption done right. (overkill, I know.)
  • SHA512-HMAC AND WHIRLPOOL-HMAC used to generate the encryption key.
  • JavaScript Blowfish encryption
  • Optional multifactor authentication with a keyfile.
  • Provides plausible deniability.
  • Optional S/KEY (one time password) multifactor authentication.

How to Install

  1. Install Apache, PHP and MySQL.
  2. Download PHP Source Code
  3. Import the SQL file to a database for password bolt
  4. Put your database information in /libs/mysql.login.php
  5. Change the keys in /libs/security.php.
    Note: The hex ones have to be exactly 768 bits and the alphanumeric ones can be anything.
  6. Change the values in the `random` table to a new random seed if you wish.
  7. Done.
