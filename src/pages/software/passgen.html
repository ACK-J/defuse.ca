Command Line Password Generator for Linux, & UNIX

This version works on any UNIX-like operating system with /dev/random. It supports the following command-line options: 

Usage: passgen <type> <optional arguments>
Where <type> is one of:
  -x, --hex				64-character hex string
  -a, --ascii				64-character ASCII string
  -n, --alpha				64-character alpha-numeric string
  -h, --help				Show this help menu
Where <optional arguments> can be:
  -q, --quick				Use /dev/urandom instead of /dev/random
  -p, --password-count COUNT		Generate COUNT passwords

How to Install on Linux

  1. Download passgen.cpp.
  2. Make sure you have g++ installed (apt-get install g++).
  3. Run: g++ passgen.cpp -o passgen
  4. The previous step created an executable file named 'passgen', move it into /usr/bin/ (You will probably need to be root to do this.)
  5. You should now be able to run the 'passgen' command to access the password generator.
