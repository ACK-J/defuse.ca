Secure Password Generator

Generating truly random passwords is hard to do, but luckily Windows and Linux come with very good random number generators! There are some good online password generators such as GRC's Perfect Passwords, but you have to trust the server not to give your passwords to someone else. This is why generating passwords from your personal computer is the safest thing to do.

Cross Platform

This one works on any *nix operating system with STL and /dev/random.

Supports the following command line parameters:
--alpha    generate a 64 character password made of numbers and letters
--ascii    generate a 64 character password made of all ASCII printable characters
--hex    generate a 64 bit password in hex

Output:

[~]$ passgen --alpha
7YYjldLtI5hGb5YcGwQb8DpAKAtQiBVdTmFjg0Hd4c8hQjXqKLuZ54MlbDQPoEtQ

How are the passwords generated?

The passwords are generated by pulling a huge random number from the operating system's random number generator. The cross platform code gets it from /dev/random, a real random number generator in unix systems. The Windows application gets it from RNGCryptoServiceProvider. To create each character in the password, it divides the huge number by the character set length. The remainder of the division is used as the index of the character in the character set. The result of the division is divided again and again to generate the next characters.
