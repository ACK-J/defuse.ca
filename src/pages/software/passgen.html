High-Security Offline Password Generator

Online password generators like GRC's Perfect Passwords can be very useful, but by using them you are trusting them not to remember your password and share it with third parties. I invite you to download one of my FREE and open source OFFLINE password generators so YOUR PC can create strong passwords with the click of a button. The Windows GUI version combines data from the Cryptographically Secure Random Number Generator (CSPRNG) built into Windows with random mouse movements to produce extremely secure passwords that are completely unique to you. The Linux version creates passwords from TRULY RANDOM noise obtained from /dev/random.

This version works on any UNIX-like operating system with /dev/random and all versions of Windows after and including Windows 2000.

Supports the following command line arguments:

  • --alpha - generate a 64 character password made of numbers and letters.

How to Install on Linux

  1. Download passgen.cpp.
  2. Make sure you have g++ installed (apt-get install g++).
  3. Run: g++ passgen.cpp -o passgen
  4. The previous step created an executable file named 'passgen', move it into /usr/bin/
  5. You should now be able to run the 'passgen' command to access the password generator.

How to Install on Windows

The easiest way to install the command line password generator on Windows is to download the executable (.exe) file and copy it into your C:\Windows\System32\ directory. After doing so, you will be able to run the password generator by typing "passgen" in the Windows command line. To open the command line, Press Start > Run > Type "cmd.exe" and hit ENTER. If the Run option isn't present in your start menu, you can open it by holding down the Windows key and pressing R.
