High-Security Offline Password Generator

Online password generators like GRC's Perfect Passwords can be very useful, but by using them you are trusting them not to remember your password and share it with third parties. I invite you to download one of my FREE and open source OFFLINE password generator so YOUR PC can create strong passwords with the click of a button. The Windows GUI version combines data from the Cryptographically Secure Random Number Generator (CSPRNG) built into Windows with random mouse movements to produce extremely secure passwords that are completely unique to you. The Linux version creates passwords from TRULY RANDOM noise obtained from /dev/random.

Windows GUI Version

This version runs on Windows XP, Windows Vista, Windows 7, Windows 2000, and Windows Server 2003+.

Linux/UNIX Command Line Version

This version works on any UNIX-like operating system with STL and /dev/random.

Supports the following command line arguments:

  • --alpha - generate a 64 character password made of numbers and letters.
  • --ascii - generate a 64 character password made of all ASCII printable characters.
  • --hex - generate a 64 bit password in hex.

How to Install

  1. Download passgen.cpp.
  2. Make sure you have g++ installed (apt-get install g++).
  3. Run: g++ passgen.cpp -o passgen
  4. The previous step created an executable file named 'passgen', move it into /usr/bin/
  5. You should now be able to run the 'passgen' command to access the password generator.
