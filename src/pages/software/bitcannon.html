What is BitCannon?

BitCannon is an easy to use secure file transfer program. It lets you send and receive files over the internet through a direct and encrypted connection. By using BitCannon, you'll be able to send files to your friends as fast as your internet connection will allow. All of your data is securely encrypted and authenticated, so you'll never have to worry about anyone else being able to see the files you send!

Features

  • Transfer any kind of file, of any size, REALLY fast!
  • MUCH faster than IM (Instant Messaging) programs!
  • No need to setup a complicated FTP servers, just a simple port forward.
  • Strong encryption keeps your files 100% secure. (AES-256, RSA-4096, HMAC-SHA256) (not fully implemented)
  • Automatic on-the-fly GZIP compression. (later)





Getting started with BitCannon is easy. All you have to do is download BitCannon and open TCP Port 59939 to your computer. Don't worry! It's not as hard as it sounds. This page, and the installation guides, will guide you through every step of the very simple installation process.

Security & Encryption

THE CRYPTO IS NOT FULLY FINISHED YET SO DON'T ASSUME TRANSFERS ARE SECURE (YET) Encryption is the key feature of BitCannon, so designed BitCannon with an extremely secure PKI system that allows BitCannon clients to transfer files between each other in a way that is fully encrypted and authenticated.

Each BitCannon client generates a 4096 bit RSA key pair the first time it starts. The central BitCannon server acts as a Certificate Authority (CA). BitCannon clients get the CA to sign their ID with their public key, so they can proove who they are to other BitCannon clients. When you transfer files with BitCannon, a secret key is exchanged between the sender and receiver using their RSA key pairs. Each client verifies the authenticity of the other's public key using the signatures provided by the CA. Once a secret key has been established, all data will be encrypted with AES-256 and authenticated with HMAC-SHA256. If you want a VERY detailed explanation of how the crypto works, see the BitCannon Cryptography page (coming soon).
