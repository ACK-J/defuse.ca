.NET Framework Benchmark Tool

This tool benchmarks your computer performance using the .NET framework. The tool comes with the following tests:

Square RootCalculates all the square roots of all numbers up to 11128654848.
List<string>Creates a massive "list" object of strings, and performs common operations on it.
GUITests the performance of WinForms by adding/removing controls from a form.
Empty LoopRuns a long loop with no body.
Empty CallsMakes many function calls to test the overhead involved with calling functions.
MathPerforms various common mathematical operations.
Big StringCreates a very large string object and calls some of .NET's string manipulation methods.
RC4Encrypts 1200000000 bytes (0x00) with RC4 using 64 zero bytes as a key.
AESEncrypts 600000000 bytes (0x00) with AES.
PrimesComputes all primes less than 10000000.
RAMPerforms memory allocation and garbage collection intensive operations.
Infinite BurnRuns the CPU at 100% until the burn is stopped.
