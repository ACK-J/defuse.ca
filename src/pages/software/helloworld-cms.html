HelloWorld! PHP CMS - Light & Secure

I've decided to release the CMS that I use for this website (ossbox.com). I designed it to be as SIMPLE as possible. Through this simplicity, we obtain a much higher level of security than that of a complex and feature packed CMS like WordPress. If you just need a simple website, this is the CMS for you.

Features

  • Script-Free CSS Menuing System. (created by http://grc.com/)
  • Doesn't requre database software.
  • Very simple & secure.
  • Highly customizable.
  • Free (open source).
  • Easily create many pages with the same layout.
  • To use it, all you'll need to know is HTML and a bit of PHP.
