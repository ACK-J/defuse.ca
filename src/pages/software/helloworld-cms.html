HelloWorld! PHP CMS - Light & Secure

This is the "CMS" we use to give each page on this website the same header and navbar. It was designed to be as simple as possible. Through this simplicity, we obtain a much higher level of security than that of a complex and feature packed CMS like WordPress. If you just need a simple website and you're familiar with HTML and CSS, HelloWorld! will do the job.

Features

  • Doesn't requre database software.
  • Very simple & secure.
  • Highly customizable.
  • Free (open source).
  • Easily create many pages with the same layout.
  • Includes the script-free CSS menuing system created by Gibson Research Corporation.
  • To use it, all you'll need to know is HTML, CSS, and a tiny bit of PHP.
