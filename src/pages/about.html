About Defuse Security

Defuse Security is Taylor Hornby's security research and development group. We try to learn how to make systems more secure, and then write software that does that. For example, we maintain one of the symmetric cryptography libraries for PHP that isn't riddled with vulnerabilities, as well as a password generator that we've taken care to keep free of side-channel vulnerabilities.

Defuse.ca's Privacy and Usability Policy

  • Assume every request made to this website is logged (including your User-Agent and IP address).
  • I use Piwik for analytics. The data it collects is hosted locally and is not shared or sold. It will also be deleted regularly. Requests with the DNT header set will not be tracked.
  • You will never need cookies to use this website.
  • There are no advertisements (or paid content) on this site.
  • I try not to include third-party content (e.g. scripts) on any of the pages of this site.
  • This website works with JavaScript turned off, and will politely tell you if it's ever needed to make one of the pages work.
