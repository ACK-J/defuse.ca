The Hit Counter

In order to count unique hits, I must be able to tell if an IP address has visited a page in the past. I do not store the IP address directly, I hash them along with the name of the page (like salt). Since the IPs are hashed using the page name as salt, I can tell if an IP has visited a specific page in the past but I cannot enumerate the IP addresses that have visited a page or correlate hits from the same IP across different pages. To do so, I would have to crack the IP hashes, which would involve creating a 4 Billion entry look-up table for each page on this site. The look-up tables would take up more than 2 Terabytes of hard drive space and would take over a month to create. Obviously, I'm not going to be doing that anytime soon. :)

If you don't want even a hash of your IP to be stored, you can enable the DNT: 1 (Do Not Track) header in your browser. If my server sees DNT: 1 in the headers your browser sends, nothing will be recorded and the hit counters will not be incremented.

You can get the source code for the hit counter here: "Privacy Aware" Unique Hit Counter for PHP.
