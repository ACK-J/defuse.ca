Awesome Desktop Calculator using Ruby's IRB

Ruby is a great programming language, but it can also be used to create an awesome desktop calculator. Ruby comes with an interactive interpreter called IRB. It lets you type ruby code and see the result of it's execution immediately. Using ruby's powerful metaprogramming abilities, IRB can be augmented to make numerical calculation easier.
