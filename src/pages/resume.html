PURE Research Project (Summer 2013 - present)

Through the summer of 2013, I worked with Dr. Michael E. Locasto on a research project exploring the feasibility of "Instruction Filters" (disabling parts of the CPU's instruction set) as a defense against code reuse exploits. This project is still a work in progress, but you can see the slides from a presentation I gave about the project here. The research was funded by a PURE Award from the University of Calgary.

The project involved:

  • Designing the defense system and modifications to the x86 architecture.
  • Implementing the architecture changes in the Bochs x86 CPU emulator.
  • Modifying the Linux kernel to support the architecture changes.
  • Using static instrumentation software (PEBIL) to add the defense system to compiled ELF binaries.
